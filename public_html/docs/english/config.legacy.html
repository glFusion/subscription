Donation Plugin

The Subscription Plugin allows site users to subscribe to additional features and to pay via PayPal. This requires the PayPal plugin to be installed and working correctly.

Donation Plugin Configuration Options

Main Settings
Option Description
Show in PayPal product catalog? Select "Yes" to have subscription products shown with other PayPal products. Select "No" if you plan to offer these products in other ways, such as static pages or blocks.
Grace Period Enter the grace period for subscription expirations. This is the number of days after a subscription expires during which the subscriber will still be able to access the additional features. Enter "0" to have the subscription terminate immediately upon expiration.
Early Renewal This is the number of days prior to a subscription's expiration when it may be renewed. You probably want to allow subscribers to renew at least a few days before their subscriptions expire.
Days before expiration to send notifications This is the number of days prior to expiration when notifications will be sent to subscribers. Only one notification will be sent per expiration; when the subscription is renewed the flag is reset to that an email will be sent before the new expiration date.
Debugging? Select "Yes" to enable verbose logging to the glFusion error log. Select "No" to turn off logging. Subscriptions, expirations and cancellations are always logged to the plugin's log.
Product Defaults
Option Description
Product is Enabled? Select "Yes" to enable new products by default. If this is "No", then new products will not be enabled unless you check the box during their creation.
Show product in block? Select "Yes" to show new products in the "Subscribe" block by default.
Product is taxable? Select "Yes" if subscription products are typically taxable.
